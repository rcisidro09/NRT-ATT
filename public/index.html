About This Tool

Upload the NRT Revenue General Ledger (QuickBooks Transaction Report) to automatically extract revenue by customer and category, then download a formatted Summary & AT&T Breakdown Excel file — ready for quarterly or annual 499Q/A reporting.

Step 1 — Upload GL File
📊
Revenue GL File Required
QuickBooks Transaction Report (XLSX)
📖 How to Export the GL File from QuickBooks Online
  1. Log in to QuickBooks Online (QBO).
  2. Go to Reports and open the Profit and Loss report.
  3. Set the date range to the quarter or period you are working on.
  4. Make sure the report is set to Cash Basis.
  5. In the P&L report, click the total amount for Revenue (above Cost of Goods Sold) — this opens the transaction detail.
  6. Click Export (top-right of the transaction list) and save as Excel (.xlsx).
  7. Upload that file here.
